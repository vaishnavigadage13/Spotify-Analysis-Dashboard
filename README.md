Spotify Analytics Dashboard
1. Project Title / Headline
🎧 Spotify Analytics Dashboard: Music Trends & Artist Performance Insights

An interactive and visually engaging Power BI dashboard designed to analyze Spotify music data, uncover artist performance trends, track popularity metrics, and explore listener engagement patterns. The dashboard enables users to gain data-driven insights into songs, artists, albums, and music consumption trends through dynamic visualizations and interactive filtering.

2. Short Description 

The Spotify Analytics Dashboard is a comprehensive Power BI report developed to analyze Spotify music data across artists, songs, albums, and engagement metrics. It helps users explore music trends, identify top-performing artists and tracks, evaluate popularity scores, and understand audience engagement through an intuitive and interactive analytical interface.

This dashboard is ideal for music analysts, record labels, marketing teams, streaming platform strategists, and music enthusiasts who want to make informed decisions based on music performance data.

3. Tech Stack

The dashboard was built using the following tools and technologies:
📊 Power BI Desktop – Primary platform used for dashboard design and data visualization.
🔄 Power Query – Utilized for data cleaning, transformation, and preparation.
🧠 DAX (Data Analysis Expressions) – Used to create calculated measures, KPIs, dynamic filtering, and advanced analytical calculations.
🎨 Custom UI Design – Spotify-inspired dark theme interface with interactive navigation buttons and media-player-style visual components.
📁 File Formats – .pbix for dashboard development and .png for dashboard previews and documentation.

4. Data Source
Source:

Spotify Music Dataset containing information about songs, artists, albums, popularity metrics, engagement scores, release dates, explicit content indicators, and album classifications.

Dataset Includes:
Song Name
Artist Name
Album Type
Release Date
Popularity Score
Engagement Score
Explicit/Non-Explicit Classification
Album Track Count
Song Position Rankings
Year and Quarter Information
Artist-Level Performance Metrics

The dataset was transformed and modeled to support trend analysis, artist comparisons, and song-level performance evaluation.

5. Features / Highlights
🎯 Business Problem
The music streaming industry generates enormous amounts of data daily. However, understanding artist performance, song popularity, engagement levels, and album trends can be challenging without a centralized analytical platform.

Key questions include:

1.Which artists have the highest-performing songs?
2.What songs generate the highest engagement?
3.How does popularity vary across albums?
4.What trends exist across different release periods?
5.How do explicit and non-explicit songs compare?
6.Which artists consistently produce successful tracks?

🎯 Goal of the Dashboard
The Spotify Analytics Dashboard was designed to:
1.Provide a centralized view of music performance metrics.
2.Enable exploration of artist and song-level insights.
3.Identify top-performing tracks and artists.
4.Analyze popularity and engagement trends.
5.Support music marketing, promotion, and content strategy decisions.
6.Deliver an interactive user experience through dynamic filtering and drill-through navigation.
